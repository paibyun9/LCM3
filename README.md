# LCM3

LCM3 – NMCC Spatial Analysis Dataset

This repository contains the dataset and code used for the spatial-statistical analysis presented in this study.

Contents:
- data.csv: Dataset of 94 WENN sites (Appendix 1)
- nmcc_ripleys_k_analysis.py: Python implementation of Ripley’s K/L-function analysis (Appendix 2)

Requirements:
- Python 3.11
- pandas
- numpy
- scipy

Usage:
python3 nmcc_ripleys_k_analysis.py

The script reproduces the results reported in Table 2.
